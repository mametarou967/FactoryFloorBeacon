# システム構成

## 機器構成

```
[担当者]
  └── MM-BLEBC8N（BLEビーコン）を携帯
        │ iBeacon advertising（UUID で個人識別）
        ▼
[Raspberry Pi]（フロア出入口に設置）
  └── BLEスキャン → 入退出イベントを記録
        │
        ▼
[ローカルストレージ]（工場内）
  └── 記録データを蓄積
        │
        ▼
[集計・確認]（未定）
```

## ビーコン仕様（MM-BLEBC8N）

| 項目 | 内容 |
|------|------|
| プロトコル | iBeacon（BLE 4.2） |
| 送信間隔 | 100ms〜10,000ms（設定可能） |
| 送信距離 | 最大約100m（出力レベル設定次第） |
| 電池 | CR2032 × 2個 |
| 電池寿命 | 数ヶ月〜約2年（設定次第） |
| 設定アプリ | SSS-825（Android専用） |

## 設計方針（検討中）

- Raspberry Pi での BLE スキャンには `bluepy` または `bleak`（Python）を使用予定
- 入退出の判定ロジックは未検討（RSSI しきい値 / 滞在時間 等）
- データフォーマットは未定
